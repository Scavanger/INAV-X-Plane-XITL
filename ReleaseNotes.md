This the first beta release for INAV XITL.

Please report all bugs and errors under [Issues](https://github.com/Scavanger/INAV-X-Plane-XITL/issues). 

### Notes
- Settings from the HITL plugin 1.X will not be transferred. 
- Requires INAV 9 with Simulator Interface V3 from https://github.com/Scavanger/INAV-XITL-Firmware

### Upgrading from HITL Plugin 1.X
- Delete folder `X-Plane 11\Aircraft\Extra Aircraft\NK_FPVSurfwing`
